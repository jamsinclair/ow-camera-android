OW Camera for Pebble

Android Companion app for the OW Camera Remote pebble app.

Download App Source on GitHub

Privacy Policy

Google Play requires me to disclose that Open Camera will record camera sensor and microphone data, which is used for the purpose of taking photos and recording videos.

Location permission is required for geotagging of photos, but this is disabled by default. If enabled, your location is encoded in the saved image and video files (and it is only used for this purpose).

The source code is released under the GPL v3 or later. The source code is adapted from the Open Camera project.
